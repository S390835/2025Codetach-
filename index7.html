Introduction

Python is a high-level, interpreted programming language known for readability, simple syntax, and quick development. It supports multiple paradigms: procedural, object-oriented and functional programming.

# Hello world in Python

print("Hello, world!")

Data type

Common built-in types: int, float, str, bool, list, tuple, set, dict. Use type() to check type.

a = 10          # int

b = 3.14        # float s = "py"       # str f = True        # bool print(type(a), type(b), type(s), type(f))

Keyword

Reserved words that have special meaning (cannot be used as identifiers). Examples: if, else, elif, for, while, def, class, return, import, from, True, False, None.

import keyword

print(keyword.kwlist)  # list of all keywords

String

Immutable sequence of characters. Use single, double or triple quotes (triple for multi-line).

s = "Hello"

print(s[0])     # H print(s[1:4])   # ell print(len(s))

f-string

name = "A" print(f"Hello {name}")

Boolean

Represents truth values: True or False. Results of comparisons and logical operations.

a = 5

b = 3 print(a > b)    # True print(a == b)   # False print(not (a < b))

List

Mutable ordered collection. Use square brackets. Supports append, pop, insert, slicing.

nums = [1,2,3]

nums.append(4) nums[0] = 0 print(nums)     # [0,2,3,4] print(nums[1:3])

Tuple

Immutable ordered collection. Created with parentheses. Useful for fixed data.

t = (1,2,3)

t[0] = 5  # Error â€” tuples are immutable

x,y,z = t   # unpacking print(x,y,z)

Array

Python lists are dynamic arrays. For numeric arrays with better performance use array module or numpy (external).

from array import array

arr = array('i', [1,2,3])  # 'i' = signed int arr.append(4) print(arr)

numpy example (requires numpy installed)

import numpy as np

a = np.array([1,2,3])

If else

Conditional execution using if, elif, else.

x = 10

if x > 0: print('positive') elif x == 0: print('zero') else: print('negative')

Operators

Arithmetic: + - * / // % **. Comparison: == != > < >= <=. Logical: and or not. Membership: in, not in.

a = 7

b = 3 print(a // b)  # integer division -> 2 print(a ** b)  # power -> 343 print((a > 5) and (b < 5))

Set

Unordered collection of unique items. Supports union, intersection, difference.

s1 = {1,2,3}

s2 = set([2,3,4]) print(s1 | s2)   # union print(s1 & s2)   # intersection s1.add(5) print(s1)

Dictionaries

Key-value mapping. Keys are usually strings or numbers. Mutable and unordered (ordered since Python 3.7 insertion order preserved).

d = {'name':'Ravi', 'age':20}

print(d['name']) d['city'] = 'Delhi' for k,v in d.items(): print(k,v)

For loop

Iterates over sequence items (list, tuple, string, range, dict keys, etc.).

for i in range(5):
print(i)

words = ['a','b','c'] for w in words: print(w)

enumerate

for idx, val in enumerate(words): print(idx, val)

While loop

Runs while a condition is true. Be careful to avoid infinite loops.

n = 3

while n > 0: print(n) n -= 1 else: print('loop finished')

Do while (Python doesn't have built-in)

Python has no native do-while. Use a while True loop with a break at the end to simulate.

= 1: break

Function

Defined with def. Can return values, have default args, *args, **kwargs, and annotations.

def add(a, b=0):
"""Return sum of a and b"""
return a + b

print(add(2,3))

lambda (anonymous) function

f = lambda x: x * 2 print(f(5))

OOPs (Object-Oriented Programming)

Core concepts: class, object, inheritance, encapsulation, polymorphism, constructor (__init__).

class Person:
def __init__(self, name, age):
    self.name = name
    self.age = age
def greet(self):
    return f"Hi, I'm {self.name}"

class Student(Person): def init(self, name, age, sid): super().init(name, age) self.sid = sid

s = Student('R', 20, 101) print(s.greet()) print(s.sid)

Class


    
ðŸ”¹ Definition:
A class defines how an object should look and behave. Instead of writing code again and again, we define a class once and reuse it.


# Defining a class
class Car:
    def __init__(self, brand, model):
        self.brand = brand      # attribute
        self.model = model      # attribute

    def show(self):             # method
        print(f"{self.brand} {self.model}")

# Creating objects (instances)
car1 = Car("Tesla", "Model S")
car2 = Car("Toyota", "Corolla")

car1.show()   # Tesla Model S
car2.show()   # Toyota Corolla
