3. Variables & Data Types

A$ variable is like a container used to store data (such as numbers, text, arrays, etc.) so we can reuse it in our program. In PHP, variables always start with a $ (dollar sign).

<?php

$greeting = "Namaste"; // string $age = 20; // integer $price = 99.99; // float $is_active = true; // boolean $fruits = ["apple","mango"]; // array ?>

Useful functions: gettype($var), var_dump($var), isset(), empty()

4. Operators

Arithmetic: + - * / % | Assignment: = += -= | Comparison: == === != !== < > <= >=

<?php

$a = 5; $b = 2; $sum = $a + $b; $isSame = ($a == '5'); // true $isIdentical = ($a === '5'); // false ?>

5. Control Structures

if / else

<?php

$score = 75; if($score >= 90){ echo "A grade"; }elseif($score >= 75){ echo "B grade"; }else{ echo "C/Fail"; } ?>

switch

<?php

$day = 3; switch($day){ case 1: echo "Monday"; break; case 2: echo "Tuesday"; break; default: echo "Other"; break; } ?>

Loops: for, while, foreach

<?php

for($i=1;$i<=5;$i++){ echo $i . "\n"; }

$fruits = ['a','b','c']; foreach($fruits as $f){ echo $f; } ?>

6. Functions

Afunctionfunction is a block of code that performs a specific task. You define it once. You can reuse it multiple times in your program.

<?php

function add($x,$y){ return $x+$y; } echo add(4,5); // 9 ?>

Variable scope: local, global (global $var), static variables (static $count).

12. Object-Oriented Programming (OOP)

<?php

class Person { public $name; private $age; public function __construct($name,$age){ $this->name = $name; $this->age = $age; } public function greet(){ return "Hello, " . $this->name; } }

$p = new Person('Anita',25); echo $p->greet(); ?>

Concepts: visibility(public/private/protected), inheritance, interfaces, traits, magic methods (__construct, __destruct, __get, __set, __call).

13. MySQL (PDO) Basic example

PDO use do careful (prepared statements)।

<?php

// PDO connection example try{ $pdo = new PDO('mysql:host=localhost;dbname=testdb;charset=utf8','root',''); $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

// Prepared statement $stmt = $pdo->prepare('SELECT * FROM users WHERE id = :id'); $stmt->execute(['id' => 1]); $user = $stmt->fetch(PDO::FETCH_ASSOC); var_dump($user); }catch(PDOException $e){ echo 'DB Error: ' . $e->getMessage(); } ?>

Never put raw user input directly into SQL — always use prepared statements.

14. Useful Built-in Functions & Tips

  • include / require / include_once / require_once — code reuse
  • header('Location: page.php') — redirect (use before output)
  • json_encode / json_decode — work with JSON
  • password_hash / password_verify — store passwords securely
  • filter_var — validation helpers (email, int)
