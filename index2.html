Definitions — Important HTML Terms

HTML

HyperText Markup Language — the standard language for creating web pages. It describes the structure of a page using elements.

Element

A building block of HTML like <p> or <a>. Elements can have attributes and content.

Attribute

Additional information added to an element, e.g., src on <img> or href on <a>.

Tag

The text in angle brackets like <div> or </div>. Opening and closing tags mark element boundaries.

Semantic HTML

Using elements that describe their meaning, like <article>, <header>, which helps accessibility and SEO.

Viewport

The visible area of a web page. The meta viewport tag helps make pages responsive on mobile devices.

Forms & Inputs

Use labels, proper types, and client-side attributes like required.

<form><label>Name</label><input type="text" required></form>

Example: Small Form

<form onsubmit="event.preventDefault(); alert('Submitted: '+document.getElementById('fn').value);">
  <label for="fn">Full name:</label>
  <input id="fn" type="text" required>
  <button type="submit">Submit</button>
</form>

Accessibility Basics

  • Use alt on images.
  • Provide labels for inputs.
  • Choose semantic elements.
  • Ensure good color contrast (this page uses bright accents).
