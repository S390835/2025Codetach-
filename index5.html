Variables

In JavaScript, a variable is like a container used to store data values (numbers, strings, objects, etc.). It allows you to give a name to a value so you can use it later in your program. . Use let, const or var. Prefer let and const in modern code.

// examples

let x = 5; const name = 'Shyam'; var old = true; // function-scoped, avoid in new code

x = x + 3; // 8
Console output

Data Types

Primitive types: number, string, boolean, null, undefined, symbol, bigint. Objects (including arrays & functions) are complex types.

non-primitive:-Array and Object

typeof 123        // 'number'

typeof 'abc'      // 'string' typeof null       // 'object' (historical JS quirk)

String

Strings are sequences of characters. Template literals (backticks) make interpolation easy.

let s = 'hello';

let t = Hello, ${s.toUpperCase()}!; console.log(t); // Hello, HELLO!
Console output

Array

An array is a special type of variable that can store multiple values in a single variable. Instead of creating many variables for each value, you can put them all inside one array.[]. They are objects with numeric keys and helpful methods.

const arr = [1,2,3,'apple'];

arr.push('mango'); console.log(arr.length); // 5
Console output

Array Methods

Common methods: map, filter, reduce, forEach, find, findIndex, slice, splice, push, pop.

// examples

const nums = [1,2,3,4]; const squares = nums.map(n => n*n); // [1,4,9,16] const evens = nums.filter(n => n%2===0); // [2,4] const sum = nums.reduce((a,b) => a+b, 0); // 10
Console output

Object

const person = {name:'Rita', age:22};

console.log(person.name); person.job = 'student';

Promise

Promises handle async operations. Use then/catch/finally or async/await.

// simple promise example

const p = new Promise((resolve,reject)=>{ setTimeout(()=> resolve('done after 1s'), 1000); });

p.then(msg => console.log(msg));

// async/await version async function waitExample(){ const res = await p; // waits for promise return res + ' (awaited)'; }
Console output
