Introduction & History

C (1972) — Dennis Ritchie, Bell Labs. Procedural, compiled language used for systems programming. Foundation for C++, Objective-C, and many modern languages.

Setup & Hello World

GCC example (Linux / WSL / Mac):

// hello.c

#include 

int main(void) { printf("Hello, world!\n"); return 0; }

// Compile: gcc hello.c -o hello // Run: ./hello

Basic Syntax

  • Statements end with ;
  • Blocks with { }
  • Comments: // line and /* block */

Data Types & Variables

Primitive types: int, char, float, double. Modifiers: short, long, signed, unsigned.

int a = 10;

char c = 'A'; float f = 3.14f; double d = 2.71828;

Operators

Arithmetic, relational, logical, bitwise, assignment, ternary.

int x = (a > b) ? a : b; // ternary

x &= 0xFF; // bitwise AND assign

Control Flow

if / else, switch, for, while, do-while, break, continue

for (int i = 0; i < n; ++i) {
printf("%d\n", i);

}

Functions

Declaration (prototype) and definition. Pass by value (C has no references).

int add(int a, int b); // prototype

int add(int a, int b) { return a + b; }

Arrays

Fixed-size, contiguous memory. Indexing starts at 0.

int arr[5] = {1,2,3,4,5};

printf("%d", arr[2]); // prints 3

Pointers

Pointers store addresses. Use & (address-of) and * (dereference).

int a = 10;

int *p = &a;  // p points to a printf("%d", p); // prints 10

Important: pointer arithmetic, NULL, void, pointer to pointer.

Strings

C strings are arrays of char terminated by '\0'.

char s[] = "hello"; // 6 bytes including '\0'

printf("%s", s);

Structs & Enums

struct Point { int x; int y; };

struct Point p = {1,2};

enum Color { RED, GREEN, BLUE };

Dynamic Memory

Use malloc, calloc, realloc, and free. Always check return value.

int *p = malloc(10 * sizeof(int));

if (!p) { perror("malloc"); exit(1); } // use p free(p);

File I/O

FILE *f = fopen("data.txt", "r");

if (f) { char buf[100]; while (fgets(buf, sizeof buf, f)) puts(buf); fclose(f); }

Preprocessor & Macros

#include <stdio.h>

#define PI 3.14159 #define MAX(a,b) ((a) > (b) ? (a) : (b))

Be careful with macros and side-effects.

Common Examples

Swap two numbers (using temporary): 
void swap(int *a, int *b) {
int t = *a; *a = *b; *b = t;

}
Factorial (iterative): 
int fact(int n) { int r = 1; for (int i = 2; i <= n; ++i) r *= i; return r; }

Cheatsheet

  • sizeof(type) — size in bytes
  • &var — address
  • *ptr — dereference
  • Strings: strlen, strcpy, strcat, strcmp from <string.h>
