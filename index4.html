1. CSS Units

Units define the size of elements in CSS. They are of two types: absolute and relative.
  • Absolute Units: px, cm, mm, in, pt, pc
  • Relative Units: %, em, rem, vw, vh, vmin, vmax
Absolute units are fixed, while relative units adapt to screen size or parent elements.

2. Flexbox

Flexbox is a one-dimensional layout system used to align and distribute space among items in a container.
  • flex-direction: row, row-reverse, column, column-reverse
  • justify-content: flex-start, flex-end, center, space-between, space-around, space-evenly
  • align-items: stretch, flex-start, flex-end, center, baseline
  • align-content: space-between, space-around, stretch, center, flex-start, flex-end
.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

3. Media Queries

Media queries are used to apply CSS rules based on device type, screen size, or orientation.
  • @media screen
  • @media print
  • @media (max-width: 768px)
@media (max-width: 768px) {
  body {
    background: lightblue;
  }
}

4. Transitions & Animations

Transitions and animations are used to create smooth visual effects in CSS.
  • Transition Properties: transition-property, transition-duration, transition-timing-function, transition-delay
  • Animation Properties: @keyframes, animation-name, animation-duration, animation-iteration-count
div {
  transition: background 0.5s ease-in-out;
}@keyframes example { 0% {background: red;} 100% {background: yellow;} }

5. CSS Grid

CSS Grid is a two-dimensional layout system for creating responsive and complex layouts.
  • grid-template-rows / columns
  • grid-gap
  • justify-items / align-items
  • grid-area
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 10px;
}
